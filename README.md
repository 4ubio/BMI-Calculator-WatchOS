# **BMI Calculator for WatchOS by Sebastián Rubio** 🧑🏽‍💻

With this app for WatchOS you can:

- calculate your BMI inserting your current height (*m*) and weight (*kg*)
- Indicates yout condition:
  1. Underweight
  2. Normal
  3. Overweight
